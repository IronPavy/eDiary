# Skolski-Dnevnik
 
Registracijom automatski postajete profesor, lista ucenika se preuzme preko JSON-a s interneta.
Šifra svakog ucenika po defaultu je 1234.
Nakon sto se ulogirate kao profesor, dodajete predmet, zatim u dodanim predmetima dodajete ucenike koje zelite.
Njih mozete ocjenjivati/brisati ocjene i sl. 
Ako se ulogirate kao ucenik vidjet cete predmete u koje ste dodani, kao i ocjene/zakljucnu/i sl.
Lozinka/avatar/podaci korisnicki se mogu mjenjati.
